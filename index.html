Data Types

UFO

  • Each row is an Item representing a UFO sighting
  • Columns are the following attributes-
    • Datetime - Sequential
    • city - Categorical
    • state - Categorical
    • country - Categorical
    • shape - Categorical
    • duration - Quantitative and Sequential
  • Column Data Type

Airports

  • Each row is an Item representing an airport and its location
  • Columns are the following attributes-
    • AIRPORT_NAME - Categorical
    • LATITUDE - Position
    • LONGITUDE - Position
  • Column Data Type

Google-Web

  • Each row is a Link between 2 nodes
  • FromNodeID represents a node
  • ToNodeId represents a node
